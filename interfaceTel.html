请求方式：http get/post

请求参数说明：

名称 必填 类型 说明
  phoneno 是 string 手机号码
  cardnum 是 string 充值金额,目前可选：1、2、5、10、20、30、50、100、200、300、500
  key 是 string 在个人中心->我的数据,接口名称上方查看

返回参数说明：

名称 类型 说明
  error_code int 返回码
  reason string 返回说明
  result string 返回结果集
